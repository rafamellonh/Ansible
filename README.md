# Ansible

> Repositorio pessoal com documentacao propria, anotacoes e exemplos praticos de Ansible.

## Sobre

Este repositorio foi criado para centralizar materiais de estudo, consultas rapidas e exemplos de automacao com Ansible.

Aqui ficam reunidos playbooks, roles, anotacoes tecnicas e referencias usadas no dia a dia.

## Conteudo

Neste repositorio voce encontra:

- exemplos de playbooks
- roles reutilizaveis
- anotacoes sobre inventario, variaveis e comandos ad-hoc
- materiais voltados para ambientes Linux, Windows e Azure

## Organizacao

Alguns dos principais conteudos estao organizados em:

- `Azure/`
- `Roles/`
- `SysAdmin/`
- `Inventory.md`
- `inventory-dynamic.md`
- `ansible-adhoc.md`
- `roles.md`
- `target-windows.md`
- `vars.md`
- `notes.md`
